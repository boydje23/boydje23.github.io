Project Description:

The Java Tic Tac Toe Game with GUI is a classic implementation of the popular Tic Tac Toe game built using Java programming language. The game provides an intuitive graphical user interface (GUI) that allows players to interact with the game board and play against each other.


Key Features:

Graphical User Interface (GUI): The game features a visually appealing GUI built using Java Swing components, providing a user-friendly interface for players to interact with.


Player vs Player Mode: Players can compete against each other in a local multiplayer mode, taking turns to make their moves on the game board.


Dynamic Game Board: The game board dynamically updates to reflect the moves made by players, providing real-time feedback and visual representation of the game progress.


Win Detection: The game includes win detection algorithms to determine when a player has achieved victory by forming a line of three consecutive symbols (X or O) horizontally, vertically, or diagonally on the game board.


Draw Detection: If all cells on the game board are filled without either player achieving victory, the game detects a draw and prompts players to start a new game.
